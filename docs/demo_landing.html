✨ Premium Features

🎨

Dark Futuristic UI

Glassmorphism cards, neon gradients, animated particle overlay, Orbitron & Rajdhani typography, and glowing hover effects.

🎮

Interactive Plotly 3D

Rotate, zoom, and pan the full robot workspace. Articulated 3R arm with coordinate frames, ground grid, and optical axis.

📷

Live Camera Projection

OpenCV-rendered 640×480 image plane with projected cube vertices, principal point crosshair, and "OUT OF VIEW" warnings.

🎛️

Real-time Controls

Joint angle sliders (θ₁, θ₂, θ₃), cube position/size, focal length, and camera tilt — all updating the scene instantly.

🎬

Auto Trajectory

Sinusoidal motion with play/pause/speed controls. Live tracking of (u, v) projection centroids over time.

📊

Analytics Dashboard

5 live metric cards, trajectory plots, position/distance metrics, Jacobian heatmap, and singular value analysis.

🚀 Quick Start

Get the premium dashboard running locally in under a minute.

# 1. Clone the repository
git clone https://github.com/Samanyu-dev/robot_vision.git
cd robot_vision

# 2. Install dependencies
pip install -r requirements.txt

# 3. Launch the dashboard
streamlit run app.py

The app will open at http://localhost:8501
