# Phase 5: All Cube Vertices in the Image Plane

## Objective

Project the full cube into the image plane by evaluating all 8 vertices and drawing the 12 cube edges.

## Geometry

The cube is centered 1.20 m along the camera optical axis for the sample pose:

```text
theta = [30 deg, 45 deg, -20 deg]
cube side length = 0.20 m
```

The world-frame cube vertices are generated by:

```text
cube_vertices_world(center, side_length)
```

The edge connectivity is stored in:

```text
src/cube.py -> CUBE_EDGES
```

## Projection Pipeline

Each vertex is projected independently:

```text
P_C = inv(T_GC) * P_G
u = fx * X_C / Z_C + cx
v = fy * Y_C / Z_C + cy
```

Only points with `Z_C > 0` are considered in front of the camera. The image-view check is:

```text
0 <= u <= 640
0 <= v <= 480
```

## Generated Artifacts

The reproducible script is:

```bash
python scripts/phase5_cube_vertices.py
```

It produces:

```text
outputs/phase5_cube_vertices.txt
outputs/phase5_cube_image.png
```

The text output contains one row per vertex with:

- world coordinates
- pixel coordinates
- visibility flag

The PNG output renders the projected cube in a 640 x 480 camera-view plot with:

- principal point marker
- edge overlay
- vertex labels
- out-of-view warning if needed
