Robot Vision Submission Report

Project: 3R manipulator with wrist-mounted camera and fixed world-space cube

1. System Setup

The system uses a 3R revolute manipulator, a pinhole camera rigidly mounted to the end-effector, and a cube fixed in the global frame. The coordinate frames are:

  • {G} Global frame
  • {B} Robot base frame
  • {E} End-effector frame
  • {C} Camera frame
  • {I} Image frame

The fixed cube center used in the submission is:

P_obj_G = [1.40, 0.60, 1.80]^T m
side length = 0.20 m

2. Transformation from World Coordinates to Image Pixel Coordinates

The derivation implemented in the project is:

P_C = T_CG P_G = inv(T_GC) P_G

s [u, v, 1]^T = K [R | t] P_G

K = [ fx  0  cx ]
    [ 0   fy cy ]
    [ 0   0  1  ]

u = fx X_C / Z_C + cx
v = fy Y_C / Z_C + cy

The camera pose is constructed from robot forward kinematics and the rigid camera mount: 

T_GC = T_GB T_BE T_EC

3. Image Coordinate Computation

For a given camera pose and object point, the project computes the image coordinate by:

  1. Transforming the world point into the camera frame
  2. Checking that the point is in front of the camera
  3. Applying the intrinsic matrix to obtain the pixel coordinate

The single-point and full-cube projection outputs are stored in:

  • outputs/phase4_image_coordinates.txt
  • outputs/phase5_cube_vertices.txt
  • outputs/phase5_cube_image.png

4. Joint Velocity Calculation

The geometric Jacobian is implemented for the 3R manipulator using: 

J_i = [ z_(i-1) x (o_E - o_(i-1)) ]
      [         z_(i-1)          ]

The project then computes the minimum-norm joint velocity: 

q_dot = pinv(J) V

The numerical Jacobian result is recorded in outputs/phase6_jacobian.txt.

5. Cube Vertex Trajectory in the Image Plane

The robot follows the prescribed trajectory: 

theta_1(t) = 30 deg + 20 deg sin(t)
theta_2(t) = 45 deg + 15 deg sin(2t)
theta_3(t) = -20 deg + 10 deg cos(t)

A specific cube vertex (+L/2, +L/2, +L/2) is projected over time into the image plane. The results are stored in:

  • outputs/phase7_trajectory.txt
  • outputs/phase7_uv_plot.png
  • outputs/phase7_uv_vs_time.png
